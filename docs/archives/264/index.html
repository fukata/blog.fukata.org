3泊4日でシンガポールに初めて行ってきた

Leave a reply
Tweet

ちょっとタイに来ており、もう少し居る必要が出てきたのでちょうど休みだったということもあり近くのシンガポールに行ってみることにしました。期間は01.31-02.03の3泊4日です。

バンコクからの行き方

AirAsiaやタイ航空、シンガポール航空など幾つかあり値段も安いやつだと往復1.5万円くらいからあったと思います。今回はシンガポール航空で行ったので3万ちょっとかかりました。

タイからシンガポールまで2時間程度で着きます。

ちなみに機内食はこんな感じ。

ホテル

知り合いの家が近くということもあり、泊まったのはOrchardにあるSingapore Hiltonです。ほとんどホテルに居ないので勿体ないですがBooking.comで探してみても聞いていた通り物価が高く、どこのホテルも大体200USD程度からでした。

SIM

荷物も受け取り、通話機能が付いてたからだと思うんですがカウンターで聞いた話ではインターネットだけで良ければStarhubで18S$のがあるみたいでした。ただ、売り場が違うとのことだったんで少し探してみたんですが見つからず。。。結局買ったのはSingTelの38S$のやつにしました。回線の品質は4Gも繋がってネット速度的にはバンコクと違って不満はありませんでした。

物価

ホテル近くのセブンイレブンでTigerビール(350ml)*2、ドリトス、ミネラルウォーター(1.5L)を買うと1600円近くしました。タイより3倍くらいするような気がします。酒飲みにはなかなか辛かったです。

食事

食事と写真が旅行の楽しみなのでここは写真多めで。チキンライス以外にも今回初めて知ったローミーやラクサなど基本的に美味しい物が多かった印象です。ラクサはスパゲッティみたいな面だったんですが、素麺みたいな細麺にしたらより美味しいんじゃないかと思いますね。スープはトムヤムクンからパンチを削ったようなココナッツ強めの味でした。

また、マリーナ・ベイ・サンズを見ながら食べるチリ・クラブもなかなかでした。

街

ガムを持ち込んではいけないというくらい徹底しているシンガポールなので街中は当然綺麗なんですが、一つ困ったところがあり、お店のトイレが綺麗じゃない。ここは中国か！？ってくらい汚い店がありました。せっかく外は綺麗なのに残念です。トイレに関してはタイのお店の方が綺麗な印象です。人件費の問題ですかね？

夜景

マリーナ・ベイ・サンズの近くは銀行のビルが多くありとても綺麗でした。三脚を持って行こうか非常に迷ったんですがリュック一つで行ったので手荷物検査などで引っかかっても面倒だなと思い、今回は諦めたんですがこの夜景を観るとやっぱり持ってくればよかったかなぁと少し後悔しました。それくらいガッツリ撮りたい風景でした。

まとめ

マーライオンやスーパーツリーなど1度訪れれば満足かなという物が多かったです。住む分には人もゆっくりしていて綺麗で良さそうですが物価も高いですし、今回はたまたま来ましたけど観光で又来たいかと言われると微妙です。
