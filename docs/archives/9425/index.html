格闘ゲームの世界大会を観戦するために東京に行ってきたので飲み歩いてきた

Leave a reply
Tweet

かなり久しぶりに東京に行く用事があったので友人と会い、飲み歩いてきた。

ホテル

HOTEL Guest1。

室内は綺麗でありがたいことに浴槽が深く、シャワーだけでなく身体の大きな自分でも浴槽に浸かることが出来た。寒い時期だったのでありがたかった。また、朝食付きで深夜まで無料で利用できるコーヒーがフロントに置いてあったのも夜遅く帰ってきた時に温かい飲み物が飲めるのも良い。

上野駅からすぐ近くということもあり飲み屋街が近いので東京についてから速攻で友人と待ち合わせし、飲み屋街に向かった。

予約はこちらから

飲み歩き

カドクラ

店内はまだお昼だというのに結構な人がいて子連れの人まで居た。金曜日の平日だというのに一体この人達は何をやってるのか。

150円からツマミが頼めるというのはありがたい。結局この日は2人で1軒各自1000円を目標に数件ハシゴすることにした。

もつ煮込み。

ハムカツ。

カドクラボール。結構甘め。

大統領

この日はハイボールで攻める。

タレが濃厚で美味い。

タレか塩かはオススメで注文。

たきおか

毎回、ハイボールで仕切り直し。

なめろう。

うなぎの肝。まさか、ここでうなぎの肝が食べられるとは思ってなかったので嬉しかった。若干冷めていたけどほろ苦くて酒がすすむ味。

文楽

この日、飲み歩き最後に寄ったのは文楽。

つくねともも。大ぶりで塩も強めで美味い。酒が進む。

Carne Tribe

同じ日の夜は牛肉を食べたいというリクエストをしていたので門前仲町にあるCarne Tribeというお店に行ってきた。

牛ハラミのステーキフリッツ。

CarneTribe盛り。

ズワイ蟹と鶏ムネ肉のアボガドフォルス。

みなせ牛のステーキとしゃぶしゃぶ。

自家製ソーセージ(豚)。

お肉のラザニア。

炊きたての土鍋ご飯とみなせ牛スネ肉のカレー。

おとん場

格闘ゲームの最終日を終えてホテルに戻った後、せっかくので飲みにまた上野の飲み屋街に繰り出した。なんとかラストオーダーギリギリに滑り込めたおとん場。

もつ焼き5本ともつ煮込み。串も90円の割には大ぶりで良い。もつ煮込みもボリューミー。

ホルモン番長

おとん場でビール2杯と串物を少し食べたので余計にお腹が空いたがめぼしい店は閉まっているかラストオーダー過ぎていたので、まだやっていた一人焼肉が食べられるホルモン番長へ。

赤身切り落としなど4種類とハイボール2杯で2300円ほど。

まとめ

普段、福岡の田舎に住んでいるので何軒もハシゴするほど店がなくてまた一人で気軽に飲める店もあまりないので、東京は一人でも気軽に飲める店が多くて便利だなと感じた。

今回行った格闘ゲームの世界大会「EVO JAPAN 2018」は来年も行う可能性が高いとのことなのでその時にはまたこの辺りに泊まって飲み歩きをしたい。
